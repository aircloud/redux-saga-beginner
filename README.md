# redux-saga-beginner

It is a demo for redux-saga, just for beginners to learning.

# Instructions

Setup

```
// clone the repo
git clone https://github.com/redux-saga/redux-saga-beginner-tutorial.git

cd redux-saga-beginner-tutorial

npm install
```

Run the demo

```
npm start
```
